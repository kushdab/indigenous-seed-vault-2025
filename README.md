# Indigenous Seed Vault 2025

## Project Overview
A specialized digital metadata archive designed to preserve and catalog the genetic heritage of indigenous Kenyan seed varieties. This tool allows researchers and conservationists to document genetic traits, local nomenclature, and regional origins (Counties) of traditional crops.

## Features
- **Rich Metadata**: Capture scientific names, local indigenous names, and specific genetic traits.
- **Trait Search**: Filter seeds by specific characteristics like 'drought-resistant' or 'high-yield'.
- **JSON Persistence**: Lightweight local database for easy portability and sharing.

## Installation
1. Ensure Python 3.8+ is installed.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the vault interface:
```bash
python vault.py
```
